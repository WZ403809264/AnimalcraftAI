# Help 帮助

## Part 1: Protocol 第1部分：协议

This software adopts UACI protocol. 这个软件采用UACI协议。

UACI was developed by imitating the UCI of chess and the UCCI of Chinese Chess. UACI是模仿国际象棋的UCI、中国象棋的UCCI来制定的。

## Part 2: Rules of Animalcraft 第2部分：野兽争霸斗兽棋规则

See www.doushouqi.net for rules. 规则详见www.doushouqi.net。

## Part 3: FEN format 第3部分：FEN格式

T1E3m1l/1C5d1/2W3p2/9/2P3w2/1D5c1/L1M3e1t r

The 16 pieces of game are represented by the following letters (upper case means the first player and lower case means the last player): 斗兽棋的16个棋子分别用以下字母表示（大写为先手方、小写为后手方）：

E(Elephant)象

L(Lion)狮

T(Tiger)虎

P(Panther)豹

W(Wolf)狼

D(Dog)狗

C(Cat)猫

M(Mouse)鼠

## Part 4: Moves Format 第4部分：着法格式

For example: 鼠→ indicates that the mouse goes to the right. 例如：鼠→表示老鼠往右走。
